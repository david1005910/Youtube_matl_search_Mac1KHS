🇰🇷 한국어 문법 검증 및 이미지 생성 테스트

테스트 결과

테스트를 실행하려면 아래 버튼을 클릭하세요.

📋 테스트 케이스 설명

  • 조사 오용 검사: 를/을, 이/가, 와/과 등의 잘못된 사용 감지
  • 어미 중복 검사: ~습니다니다, ~니다습니다 등의 중복 어미 수정
  • 띄어쓰기 오류: 기본 인사말, 감사 표현의 잘못된 띄어쓰기 수정
  • 맞춤법 오류: 되/돼, 됬/됐 등의 흔한 맞춤법 오류 수정
  • 한영 혼용: 한글과 영문 사이 적절한 띄어쓰기 추가
  • 정상 텍스트: 올바른 텍스트는 변경하지 않음 확인

🎨 실제 이미지 생성 테스트 방법

  1. 메인 앱에서 "🎨 이미지 생성" 버튼 클릭
  2. 다음 중 하나의 테스트 프롬프트 입력:
    • "안녕하세요를을 쓴 예쁜 간판" (문법 오류)
    • "감 사합니다 라고 적혀있는 카페 메뉴판" (띄어쓰기 오류)
    • "한국어 텍스트가 아름답게 디자인된 포스터" (정상)
  3. "한국어 텍스트 문법 검증" 체크박스 확인
  4. "🎨 이미지 생성하기" 버튼 클릭
  5. 프롬프트가 자동으로 수정되는지 확인
  6. 생성된 이미지의 한국어 텍스트가 올바른지 확인
