# 📖 유튜브 소재 발굴 도구 — 사용 설명서

키워드로 **바이럴 소재 영상**을 찾고 → 댓글을 AI로 분석하고 → 대본 목차를 만들고 → 장면 이미지와 영상까지 생성하는 전체 흐름을 한 화면에서 처리합니다.

> 빠른 시작: `python3 server.py` 실행 후 브라우저에서 `http://localhost:8765` 접속 → API 키 입력 → 키워드 검색.

## 1. 시작하기

### 권장: 로컬 서버 모드
API 키가 `.env`에 저장되고, AI 이미지·영상 생성 등 모든 기능이 동작합니다.

```bash
python3 server.py        # http://localhost:8765 접속
```

### 간편: 브라우저 직접 실행
검색·분석·대본만 빠르게 쓸 때. (키는 브라우저 세션에만 저장됨)

```bash
open index.html
```

## 2. API 키 설정

화면 상단 패널에 키를 입력하면 됩니다 (서버 모드는 자동 저장).

| 키 | 용도 | 필수 |
|----|------|:---:|
| **YouTube Data API v3** | 영상/댓글 검색 | ✅ |
| **Gemini API** | 댓글 분석·대본·이미지 | ✅ |
| TranscriptAPI | 자막 추출 위젯 | 선택 |
| Stability / xAI(Grok) / imgbb | 이미지·영상 생성 | 선택 |

- YouTube 키: [Google Cloud Console](https://console.cloud.google.com) → YouTube Data API v3 활성화 → API 키 발급
- Gemini 키: [Google AI Studio](https://aistudio.google.com/app/apikey) → 무료 키 생성

## 3. 영상 검색 & 바이럴 비율

### 검색 방법
1. **키워드** 입력 (한국어 우선 검색)
2. 필터 선택:
   - **영상 형태**: 숏폼(3분 미만) / 롱폼 구간(3~5분 … 15분 이상)
   - **최소 바이럴 비율**: 50% ~ 5,000% (기본 200%)
   - **검색 기간**: 최근 3개월 ~ 2년
   - **결과 수**: 10 ~ 50개
3. **검색** 클릭 → 바이럴 비율 높은 순으로 정렬

### 바이럴 비율이란?
> **바이럴 비율 = (조회수 ÷ 구독자 수) × 100**

구독자 규모와 무관하게 **소재 자체의 폭발력**을 측정합니다.

| 비율 | 의미 |
|------|------|
| 200% 이상 | 소재력 검증된 영상 |
| 500% 이상 | 강력한 바이럴 소재 |
| 1,000% 이상 | 알고리즘 폭발 영상 |

각 결과 카드: 제목 · 조회수 · 구독자 수 · 비율 게이지 · **[💬 댓글 수집 및 AI 분석]** 버튼.

## 4. AI 댓글 분석

영상 카드의 **[💬 댓글 수집 및 AI 분석]** 클릭 → 상위 댓글을 모아 Gemini가 분석합니다.

분석 결과:
- ✅ **시청자 반응** — 열광하는 포인트
- ❓ **Pain Points** — 아쉬운 점 & 추가 궁금증
- 🏷️ **자주 언급된 키워드 TOP 5** — 언급 맥락 포함
- 🚀 **바이럴 3대 요소** — CTR / AVD / 참여도 점수 & 개선 팁
- 🎬 **추천 주제 키워드 5개** — 기획 방향 + 예상 제목

## 5. AI 대본 목차 생성

추천 주제 5개 중 **1개 선택** → **분량 선택** → 대본 목차 생성.

| 분량 | 내용 |
|------|------|
| 짧게 (~500자) | 핵심만 |
| 보통 (~1,000자) | 표준 구성 |
| 상세 (~2,000자) | 설명 포함 |
| 완전 대본 (~3,000자) | 실제 말할 수 있는 형식 |

생성 항목: 영상 제목 후보 3개 · 썸네일 컨셉 · 오프닝 훅 · 챕터별 대본 · 클로징 & CTA · 제작 팁.

## 6. AI 이미지 생성

대본을 바탕으로 **장면별 이미지 카드**를 만듭니다. (보라색 🎨 위젯 → **AI 이미지 생성** 모달)

### 모델 선택
- 🌈 **Imagen** (기본, Gemini)
- 🎨 **Stable Diffusion** (Stability 키 필요)
- 🌸 **Pollinations** (무료, 키 불필요)

### 카드 편집 (장면별)
- **🎥 컷 묘사** / **📹 촬영·편집 방향** / **💬 자막 텍스트**
- **🤖 AI 자동 채우기** — 위 3가지를 Gemini가 한국어로 자동 작성
- **🎨 이미지 생성** — 해당 카드 이미지 생성

### 글자 없는 이미지 + 한글 자막
- 이미지는 **글자/워터마크 없이 깨끗하게** 생성됩니다 (프롬프트의 텍스트 지시 자동 제거).
- 자막은 생성 후 **Canvas로 따로 입힙니다**. 영어 자막은 **한글로 자동 번역**되어 한글 폰트로 렌더링됩니다.
- **🔄 자막 재적용** — 자막 텍스트를 수정한 뒤 다시 입힐 때.

## 7. AI 영상 생성 (Phosphene · Grok)

생성한 카드 이미지를 **영상(image→video)**으로 만듭니다.

### Phosphene (로컬 · 권장) — Apple Silicon Mac
클라우드·API 키 없이 맥에서 로컬로 동작합니다 (LTX-2 / MLX).

**준비**
1. [Pinokio](https://pinokio.computer)에서 **Phosphene** 설치·실행 (패널 `http://127.0.0.1:8198`)
2. `.env`의 `PHOSPHENE_URL` (기본값으로 자동 연결)
3. (합치기용) `brew install ffmpeg`

**카드별 영상 생성**
1. 이미지가 생성된 카드의 **🎬 Phosphene 영상 생성** 클릭
2. **⚙️ 옵션** — 해상도(736×416 ~ 1280×720·세로) · 길이(5/7/10초) · 품질(quick~high) · 모션 프롬프트
3. 완료 시 카드 안에서 **재생·다운로드**

**전체 일괄 + 합치기**
- Grok 영상 위젯의 **🎬 전체 카드 영상 생성 + 합치기 (Phosphene)** → 모든 카드를 순차로 영상화 후 하나의 MP4로 합쳐 다운로드

> ⏱️ 로컬 렌더라 클립당 약 5분(736×416 기준), 한 번에 하나씩 순차 처리됩니다. 빠른 미리보기는 **736×416 / quick** 권장.

**✅ 검증 상태** — 실제 앱 화면에서 end-to-end 동작 확인 완료 (2026-06-24):
- 카드별 영상 생성: 이미지 업로드 → i2v 렌더 → 카드 내 재생까지 정상
- 전체 일괄 + 합치기: 카드 2개 순차 렌더 → 하나의 MP4(10초)로 합쳐 다운로드까지 정상

### Grok (클라우드) — xAI
- 카드의 **🎬 Grok 영상 생성** (xAI 키 필요). 이미지→영상은 imgbb 공개 URL 경유.

## 8. 자막 굽기 (Remotion)

영상 파일에 자막을 입혀 MP4로 굽습니다. (주황색 🎞️ 위젯)

**준비**
```bash
cd remotion && npm install && npm start   # http://localhost:8766
```

**사용**
1. 영상 업로드 / 자막 가져오기(TranscriptAPI)
2. (선택) Gemini 번역으로 2중 자막
3. **🎞️ 자막 렌더링 시작** → 자막이 박힌 MP4 다운로드

## 10. 자주 묻는 질문

**Q. API 키가 자꾸 사라져요.**
→ `python3 server.py`(로컬 서버 모드)로 실행하면 `.env`에 저장됩니다. 브라우저 직접 실행은 세션에만 저장됩니다.

**Q. 이미지에 이상한 영어 글자가 박혀요.**
→ 최신 버전은 프롬프트의 텍스트 지시를 제거해 글자 없는 이미지를 생성합니다. 브라우저를 **강제 새로고침**(Cmd+Shift+R) 후 다시 생성하세요.

**Q. 자막이 영어로 나와요.**
→ 자막 텍스트 필드에 입력한 내용이 입혀집니다. 영어는 한글로 자동 번역됩니다. 이미 만든 이미지는 **🔄 자막 재적용**을 누르세요.

**Q. Phosphene 영상 버튼이 "오프라인"이에요.**
→ Pinokio에서 Phosphene 패널이 실행 중인지 확인하세요(`http://127.0.0.1:8198`). 포트가 다르면 `.env`의 `PHOSPHENE_URL`을 수정합니다.

**Q. 전체 합치기(concat)가 실패해요.**
→ `brew install ffmpeg`로 ffmpeg를 설치하세요.

**Q. YouTube 검색이 안 돼요 / 할당량 초과.**
→ YouTube Data API v3는 하루 10,000유닛 무료입니다. 검색 1회당 약 200~300유닛 소모되니 키를 확인하거나 다음 날 재시도하세요.
